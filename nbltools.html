Overview

NBLtools is a comprehensive preprocessing pipeline for diffusion MRI data. It provides automated tools for data quality control and correction, ensuring your diffusion data is optimally prepared for analysis.

Key Features

  • Denoising for noise reduction
  • Gibbs ringing correction
  • Optibet for optimal brain masking
  • Topup for susceptibility-induced distortion correction
  • Eddy current and motion correction
  • Comprehensive quality control reporting

System Requirements

NBLtools requires the following software dependencies to be installed:

ANTs

Advanced Normalization Tools for image registration and segmentation

FSL

FMRIB Software Library for brain imaging analysis

TORTOISE 4.0

Tolerably Obsessive Registration and Tensor Optimization Indolent Software Ensemble

Memory (RAM)

Minimum: 8GB
Recommended: 16GB+ for large datasets

Important: All software dependencies (ANTs, FSL, TORTOISE) must be properly installed and available in your PATH environment variable for Linux/WSL and macOS bash.

Docker Container

For Windows, Linux, and macOS (Intel) users, we provide a Docker container with all dependencies pre-installed. This is the easiest way to get started with NBLtools.

Running NBLtools with Docker

Interactive Docker Command

Run NBLtools interactively with all dependencies included:

docker run -it -v :/data rstones/dwi-preproc bash

Replace /path/to/your/data with the path to your diffusion MRI data directory

Docker Installation

  1. Install Docker Desktop from docker.com
  2. Pull the NBLtools Docker image (instructions on GitHub)
  3. Run the command above to start processing your data
Note: macOS Apple Silicon (M-series) users may experience compatibility issues with the Docker container. We recommend using the native installation with the required dependencies.

Getting Started

To use NBLtools:

  1. Visit the GitHub repository for installation instructions
  2. Either install dependencies manually (ANTs, FSL, TORTOISE 4.0) or use Docker
  3. Follow the setup guide in the repository README
  4. Prepare your diffusion MRI data according to the documentation
  5. Run the preprocessing pipeline using the provided scripts
