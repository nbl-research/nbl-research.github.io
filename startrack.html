Overview

StarTrack is a program to analyse diffusion imaging data using DTI, spherical deconvolution (SD) and tractography. At its core, it computes diffusion tensors or fibre orientation distributions (FOD) and performs whole-brain tractography using different algorithms to map the human white matter anatomy.

StarTrack also allows you to display, manipulate and rotate fibre orientation distributions in large brain regions to help understand, calibrate and fine-tune SD parameters.

Key Features

  • Diffusion Tensor Imaging (DTI) analysis
  • Spherical deconvolution (SD) with FOD computation
  • Whole-brain tractography with multiple algorithms
  • Interactive FOD visualization and manipulation
  • Parameter calibration and fine-tuning tools
  • Compatible with TrackVis, FSL and all software supporting .trk format

System Requirements

This version of StarTrack is based on standalone MATLAB Runtime Compiler 2023b.

Note: A new Python-based version of StarTrack is currently in preparation.

Memory (RAM)

Minimum: 8GB
Recommended: >16GB for analysing large datasets (e.g. HCP and similar)

Operating System

Linux (x86), Windows (x86), macOS (x86), or macOS (Apple Silicon)

Runtime

MATLAB Runtime Compiler 2023b (installation instructions provided with download)

Getting Started

To install and run StarTrack:

  1. Install MATLAB Runtime Compiler 2023b (installation instructions included in the download)
  2. Download the appropriate version of StarTrack for your operating system
  3. Extract the archive to a folder on your system
  4. Follow the instructions in the README file on how to start the software in a terminal
  5. Refer to the user manual for detailed usage instructions and examples
